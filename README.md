# twitter clone

i appen så kommer man til å få mullighten til å lage profile og komminisere med andre sende gifs og like andre sine poster . 
det er samme som twitter der man kna se hva som er populær , søke andre sine navn og post . 

## MVP (Minimum Viable Product)


### hva skal den inholde

- kunne lage bruker
- kunne legge ut post
- kunne legge ut kommentarer og like,dislike og save post
- kunne sende meldinger
- kunne designe profilen sin
- UI og UX


### hva skal jeg demonstrere

- backend logikk
- database bruk
- Brukerautentisering
- supabase

### teknologier som jeg bruker

- next.js
- react
- supabase
- PostgreSQL
- supabase auth
- supabase storage
- TypeScript

## måten jeg viser kompetanse

på utvikling så viser jeg måten frontend og backend kommuniserer. vise forskjellige auth som google og kansje github
